Getting Started

5 分钟接入 MaaS 统一 API

你可以使用 OpenAI SDK 不改业务逻辑地迁移到 MaaS 平台，仅需替换 base_url 与 api_key。

推荐先使用测试项目和限额 Key，验证路由策略与成本后再切生产环境。

1. 安装 SDK

bash 
pip install openai
bash 
npm i openai
bash 
go get github.com/openai/openai-go

2. 发送第一个请求

python 
from openai import OpenAI

client = OpenAI(
  api_key="maas_xxx",
  base_url="https://api.maas-platform.com/v1"
)

resp = client.chat.completions.create(
  model="gpt-4o",
  messages=[{"role": "user", "content": "你好"}],
)

print(resp.choices[0].message.content)
如果你启用了企业白名单 IP，记得将服务器出口地址加入控制台安全策略。
Security

鉴权方式

平台支持 API Key 与企业 SSO，两种模式可并行使用。

方式适用对象优点注意事项
API Key服务到服务调用简单直连建议 90 天轮转
JWT + SSO控制台人机登录统一身份治理依赖企业 IdP
API Reference

Chat Completions

POST /v1/chat/completions

请求头

  • Authorization: Bearer maas_xxx
  • Content-Type: application/json

关键参数

  • model: 目标模型 ID
  • messages: 对话消息列表
  • stream: 是否使用 SSE
json 
{
  "model": "gpt-4o",
  "stream": true,
  "messages": [
    { "role": "user", "content": "解释一下RAG" }
  ]
}
