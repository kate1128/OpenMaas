## 核心术语表

| **术语** | **定义** | **使用边界** | **页面/模块映射** |
| --- | --- | --- | --- |
| **账期** | 平台进行收入、成本、差异核算的结算周期，默认按自然月统计，如 `2026-05`。 | 所有账单展示、导出、对账、出账动作都必须显式绑定账期。 | 账单管理四个 Tab、账单导出文件名、对账邮件摘要 |
| **总账视图** | 面向平台运营的全局账务总览，聚焦平台级收入、成本、毛利、毛利率及趋势。 | 只看平台汇总口径，不展开单租户逐笔处理。 | 管理后台 → 账单管理 → 总账视图 |
| **租户分账** | 以租户为结算对象的账单聚合视图，用于查看单租户在指定账期内的收入、成本、毛利与差异。 | 用于租户级核对、导出、跟进，不替代平台总账。 | 管理后台 → 账单管理 → 租户分账 |
| **对账差异** | 平台内部账务口径与厂商账单口径之间的金额偏差及其处理记录。 | 仅用于差异核查、归因、确认、修正，不直接代表最终出账金额。 | 管理后台 → 账单管理 → 对账差异 |
| **出账记录** | 针对租户或账期已经生成的正式账单流水与下载记录。 | 只记录已进入出账流程或已完成出账的账单快照。 | 管理后台 → 账单管理 → 出账记录 |
| **差异阈值** | 判断账单能否直接生成出账的可配置金额阈值。 | 当差异金额超过阈值时，必须先处理差异，再允许生成账单。 | 出账确认弹窗、账单生成按钮禁用规则 |
| **毛利** | 平台在指定核算范围内的 `收入 - 厂商成本`。 | 可在平台总账与租户分账两个层级展示，但口径必须明确。 | 总账 KPI 卡片、租户分账列表 |
| **毛利率** | 毛利占收入的比例，计算公式为 `gross_profit / total_revenue`。 | 当收入为 0 时展示为 `--`，不得错误显示为 0%。 | 总账趋势与风险提示 |
| **对账状态** | 差异或租户账单当前所处的核对阶段，如 `待核实`、`已存疑`、`已确认`、`已修正`。 | 用于驱动按钮显隐、禁用和审计记录，不等同于出账状态。 | 租户分账、对账差异弹窗与列表 |
| **出账状态** | 正式账单生成流程状态，如 `待出账`、`对账中`、`已出账`。 | 仅作用于出账流水，不用于描述差异处理进度。 | 出账记录列表与账单生成流程 |

## 使用约定

1. PRD 中涉及账单管理名词时，优先引用本术语表，不再在正文重复定义。
2. 原型设计文档、HTML 原型、接口文档、测试用例中的账单域文案必须与本表一致。
3. 若新增账单域名词，先更新本文件，再同步业务文档与页面原型。
