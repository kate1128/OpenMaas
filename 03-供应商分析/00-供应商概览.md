## 5. 关键差异汇总

### 5.1 鉴权方式差异

| 供应商 | 鉴权方式 | Header 格式 |
|--------|---------|------------|
| OpenAI / DeepSeek / Moonshot | Bearer Token | `Authorization: Bearer sk-xxx` |
| 阿里云百炼 | Bearer Token | `Authorization: Bearer sk-xxx` |
| 智谱 AI | JWT（需本地签名） | `Authorization: Bearer <jwt>` |
| 字节豆包 | Bearer Token | `Authorization: Bearer ark-xxx` |
| 百度文心 | Access Token（OAuth） | `?access_token=xxx` 或 Header |
| Google Gemini | API Key / OAuth | `?key=xxx` 或 `Authorization: Bearer <oauth>` |

### 5.2 流式输出格式

所有主流供应商均支持 `stream: true` 的 SSE 流式输出，格式基本统一：
```
data: {"id":"...","choices":[{"delta":{"content":"..."}}]}
data: [DONE]
```

百度文心流式格式略有不同，`is_end` 字段替代 `[DONE]`。
