## 2. API 接入

### 2.1 鉴权：OAuth 两步认证

百度文心使用 **OAuth 2.0 Client Credentials** 获取 Access Token：

**Step 1：获取 Access Token（有效期 30 天）**
```http
POST https://aip.baidubce.com/oauth/2.0/token
  ?grant_type=client_credentials
  &client_id=<API_Key>
  &client_secret=<Secret_Key>
```

响应：
```json
{
  "access_token": "24.abc123...xyz.2592000.1716200000.xxxx",
  "expires_in": 2592000,
  "token_type": "Bearer"
}
```

**Step 2：用 access_token 调用 API**
```http
POST https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop/chat/completions_pro
  ?access_token=24.abc123...xyz
Content-Type: application/json
```

> Access Token 有效期 30 天，需要实现自动刷新。

### 2.2 API Endpoint 按模型分路由

百度文心没有统一 endpoint，每个模型有独立的 URL 路径：

| 模型 | Endpoint 路径（追加在 base 之后） |
|------|--------------------------------|
| ERNIE-4.0 | `/chat/completions_pro` |
| ERNIE-4.0-Turbo | `/chat/ernie-4.0-turbo-8k` |
| ERNIE-3.5 | `/chat/completions` |
| ERNIE-Speed-128K | `/chat/ernie-speed-128k` |
| ERNIE-Lite-8K | `/chat/ernie-lite-8k` |
| ERNIE-Tiny-8K | `/chat/ernie_tiny` |
| Embedding v1 | `/embeddings/embedding-v1` |
| bge-large-zh | `/embeddings/bge_large_zh` |

Base URL：
```
https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop
```

## 3. 调用格式

### 3.1 Chat 请求（自有格式，与 OpenAI 不同）

```json
{
  "messages": [
    {"role": "user",      "content": "你好，介绍一下百度"},
    {"role": "assistant", "content": "百度是一家互联网公司..."},
    {"role": "user",      "content": "它的主要产品是什么？"}
  ],
  "stream": false,
  "temperature": 0.7,
  "top_p": 0.9,
  "penalty_score": 1.0,
  "system": "你是一个有帮助的助手",
  "user_id": "uid_12345"
}
```

**与 OpenAI 的差异：**
- ❌ 没有 `model` 字段（模型在 URL 中指定）
- ✅ `system` 字段在请求体顶层（不在 messages 里）
- ⚠️ messages 中必须保证 `user/assistant` 交替出现
- ⚠️ 不支持 messages[0] 为 assistant

### 3.2 Chat 响应格式

```json
{
  "id": "as-abc123",
  "object": "chat.completion",
  "created": 1716200000,
  "result": "百度的主要产品包括百度搜索、百度地图...",
  "is_truncated": false,
  "input_tokens": 30,
  "output_tokens": 150,
  "usage": {
    "prompt_tokens": 30,
    "completion_tokens": 150,
    "total_tokens": 180
  }
}
```

**与 OpenAI 的差异：**
- 答案在 `result` 字段（非 `choices[0].message.content`）
- 没有 `choices` 数组结构

### 3.3 流式输出

```
data: {"id":"as-abc","object":"chat.completion","result":"百度","is_end":false}
data: {"id":"as-abc","object":"chat.completion","result":"的主","is_end":false}
data: {"id":"as-abc","object":"chat.completion","result":"要产品","is_end":false}
data: {"id":"as-abc","object":"chat.completion","result":"","is_end":true,"usage":{...}}
```

**与 OpenAI 的差异：**
- 使用 `is_end: true` 替代 `[DONE]`
- `result` 是增量文本（非 delta.content）

### 3.4 Function Calling

```json
{
  "messages": [{"role": "user", "content": "查询北京天气"}],
  "functions": [
    {
      "name": "get_weather",
      "description": "获取城市天气",
      "parameters": {
        "type": "object",
        "properties": {
          "city": {"type": "string"}
        },
        "required": ["city"]
      },
      "responses": {
        "type": "object",
        "properties": {
          "temperature": {"type": "string"}
        }
      }
    }
  ]
}
```

> 使用 `functions` 字段（非 OpenAI 的 `tools`）；响应中使用 `function_call` 字段。

### 3.5 Embedding

```json
// 请求
{
  "input": ["Hello world", "你好世界"]
}
// 响应
{
  "id": "xxx",
  "object": "embedding_list",
  "data": [
    {"object": "embedding", "index": 0, "embedding": [0.0023, ...]},
    {"object": "embedding", "index": 1, "embedding": [-0.0011, ...]}
  ],

}
```

## 7. MaaS 接入评估

### 7.1 Adapter 复杂度：**高**

需要实现完整的格式转换适配器：

```python
class WenxinAdapter:
    """百度文心 OpenAI 格式适配器"""
    
    BASE_URL = "https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop"
    
    MODEL_ENDPOINTS = {
        "ernie-4.0":         "/chat/completions_pro",
        "ernie-4.0-turbo":   "/chat/ernie-4.0-turbo-8k",
        "ernie-3.5":         "/chat/completions",
        "ernie-3.5-128k":    "/chat/ernie-3.5-128k",
        "ernie-speed":       "/chat/ernie_speed",
        "ernie-speed-128k":  "/chat/ernie-speed-128k",
        "ernie-lite":        "/chat/ernie-lite-8k",
    }
    
    def __init__(self, api_key: str, secret_key: str):
        self.api_key = api_key
        self.secret_key = secret_key
        self._access_token = None
        self._token_expire = 0
    
    def get_access_token(self) -> str:
        if time.time() < self._token_expire - 300:
            return self._access_token
        resp = requests.post(
            "https://aip.baidubce.com/oauth/2.0/token",
            params={
                "grant_type": "client_credentials",
                "client_id": self.api_key,
                "client_secret": self.secret_key
            }
        )
        data = resp.json()
        self._access_token = data["access_token"]
        self._token_expire = time.time() + data["expires_in"]
        return self._access_token
    
    def openai_to_wenxin(self, openai_req: dict) -> tuple[str, dict]:
        """转换 OpenAI 格式请求为文心格式"""
        model = openai_req.get("model", "ernie-3.5")
        endpoint = self.MODEL_ENDPOINTS.get(model, "/chat/completions")
        url = f"{self.BASE_URL}{endpoint}?access_token={self.get_access_token()}"
        
        # 提取 system 消息
        messages = openai_req.get("messages", [])
        system = next((m["content"] for m in messages if m["role"] == "system"), None)
        non_system = [m for m in messages if m["role"] != "system"]
        
        wenxin_req = {
            "messages": non_system,
            "stream": openai_req.get("stream", False),
            "temperature": openai_req.get("temperature", 0.7),
            "top_p": openai_req.get("top_p", 0.9),
        }
        if system:
            wenxin_req["system"] = system
        
        return url, wenxin_req
    
    def wenxin_to_openai(self, wenxin_resp: dict) -> dict:
        """转换文心响应为 OpenAI 格式"""
        return {
            "id": wenxin_resp.get("id", ""),
            "object": "chat.completion",
            "created": wenxin_resp.get("created", int(time.time())),
            "model": "wenxin",
            "choices": [{
                "index": 0,
                "message": {
                    "role": "assistant",
                    "content": wenxin_resp.get("result", "")
                },
                "finish_reason": "stop"
            }],
            "usage": wenxin_resp.get("usage", {})
        }
```

### 7.2 接入配置

```yaml
vendor: wenxin
auth_type: oauth_client_credentials
client_id_env: WENXIN_API_KEY
client_secret_env: WENXIN_SECRET_KEY
token_refresh_days: 29              # 30天过期，提前1天刷新
adapter: wenxin_custom              # 需要专用 Adapter
timeout: 60s
```

### 7.3 注意事项

1. **完全不兼容 OpenAI 格式**，必须实现完整转换层
2. **OAuth Token 管理**：30天有效期，需要持久化存储并自动刷新
3. **每个模型独立 URL**：需要维护模型名 → endpoint 路径的映射
4. **messages 格式严格**：user/assistant 必须交替，不能以 assistant 开头
5. **联网搜索收费**：开启 `enable_web_search` 会产生额外费用
6. **适用场景**：有国内合规要求（数据本地化）、需要百度知识图谱、政务/金融场景

## 8. 常见错误码

| 错误码 | 含义 | 处理建议 |
|:------:|-----|---------|
| 110 | Access Token 无效 | 重新获取 Token |
| 111 | Access Token 过期 | 刷新 Token |
| 336000 | 请求参数错误 | 检查 messages 格式 |
| 336100 | 超出 API 速率限制 | 退避重试 |
| 336501 | 模型服务不可用 | 稍后重试 |
| 336502 | 内容违规 | 检查 prompt 内容 |
